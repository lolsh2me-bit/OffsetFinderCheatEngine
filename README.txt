FOX OFFSET FINDER — СБОРКА EXE

Вариант 1: без установки Python на свой компьютер
1. Создайте пустой репозиторий GitHub.
2. Загрузите в него всё содержимое этой папки, включая .github.
3. Откройте вкладку Actions.
4. Выберите Build Windows EXE и нажмите Run workflow.
5. После завершения скачайте артефакт FoxOffsetFinder-Windows.
6. Внутри будет FoxOffsetFinder.exe — Python для запуска не нужен.

Вариант 2: локальная сборка
1. Установите Python 3.11+ для Windows.
2. Запустите build.bat.
3. EXE появится в папке dist.

Примечания:
- Программа предназначена для 64-битной Windows.
- Она только читает память процесса и ищет кандидаты указателей.
- Найденный кандидат не гарантирует стабильную pointer chain.
- Используйте на собственных программах и офлайн-играх без античита.
